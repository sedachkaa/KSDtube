# 🎵 KSDtube

**KSDtube** — это десктопное приложение для скачивания музыки (MP3) и видео (MP4) с YouTube с автоматическим добавлением метаданных (обложка, теги), поддержкой плейлистов, обрезкой фрагментов и встроенным DPI-обходчиком Zapret.

![Главное окно](screenshot.png) <!-- добавьте скриншот позже -->

## ✨ Возможности

- 🎵 **Скачивание MP3** с максимальным качеством (192 кбит/с) и автоматическим встраиванием обложки.
- 🎬 **Скачивание MP4** в любом качестве (до 4K, если доступно).
- 📂 **Поддержка плейлистов** — скачивание всех видео из плейлиста.
- ✂️ **Обрезка** — удаление вступлений, концовок или любых фрагментов по времени.
- 🏷️ **Метаданные** — название, исполнитель, альбом, год, жанр, обложка (встраиваются прямо в файл).
- 🔍 **Поиск** — находите видео по ключевым словам прямо из приложения.
- 🛡️ **Обход блокировок (Zapret)** — встроенный инструмент для обхода DPI (помогает скачивать даже в РФ).
- 🌙 **Тёмная тема** — приятный для глаз интерфейс.
- 💾 **История загрузок** — SQLite-база с сохранением всех скачанных файлов.
- 📁 **Простая настройка** — выбор папки для сохранения, автообновление истории, настройка автозапуска Zapret.

## 🚀 Установка

### Готовый установщик (рекомендуется)
1. Скачайте файл `KSDtube_Setup.exe` из раздела **Releases**.
2. Запустите установщик, следуйте инструкциям.
3. После установки на рабочем столе появится ярлык KSDtube.

### Портативная версия (ZIP-архив)
1. Скачайте архив `KSDtube_portable.zip`.
2. Распакуйте в любую папку.
3. Запустите `KSDtube.exe`.

## 🖥️ Системные требования

- **Windows 10/11** (64-bit).
- **Поддержка аппаратного ускорения** (для Zapret) – необязательно.
- **Интернет** – для скачивания с YouTube.

## 🎮 Как пользоваться

1. **Вставьте ссылку** на YouTube видео или плейлист.
2. **Нажмите «Предпросмотр»** – увидите название, канал, длительность и обложку.
3. **Выберите формат** (MP3 или MP4) и качество (для видео).
4. **Опционально** – включите «Обрезать» и укажите время (например, `1:30` — `4:20`).
5. **Нажмите «Скачать»** – файл появится в выбранной папке (по умолчанию `Downloads/KSDtube`).

### Особенности для пользователей из РФ
- Если YouTube блокирует скачивание, нажмите на статус `🛡️ Zapret: неактивен` в правом нижнем углу.  
  Откроется консоль, запустится `service.bat` (требует прав администратора). После этого обход включится.
- Статус обновляется автоматически каждые 5 секунд.
- При первом запуске Zapret может потребоваться установить драйвер WinDivert – разрешите.

## 🔧 Настройки

- **Папка сохранения** – кнопка `📁 Выбрать папку`.
- **Автозапуск Zapret** – настройка в диалоге Zapret (открывается кликом по статусу).
